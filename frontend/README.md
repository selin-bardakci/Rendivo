# Rendivo Frontend (frontend/)

This folder contains a Next.js + TypeScript frontend scaffold for the Rendivo project.

To run:

1. cd frontend
2. npm install
3. create a `.env.local` with required NEXT_PUBLIC_* variables (see top-level README)
4. npm run dev

Files:
- `pages/` — index, login, dashboard
- `lib/` — api, auth, firebase, fcm
- `components/` — Layout, Nav, ProtectedRoute
